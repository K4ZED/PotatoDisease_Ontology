# Potato Disease Ontology - Progress 1

## Deskripsi Proyek

Sistem Diagnosa Penyakit Kentang Berbasis Ontologi ini bertujuan untuk membantu pengguna dalam mengidentifikasi penyakit pada tanaman kentang. Dengan memanfaatkan ontologi, sistem ini menyediakan informasi detail tentang berbagai penyakit dan memungkinkan diagnosis berdasarkan gejala.
